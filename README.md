**Juntoo** is the all-in-one financial companion for your family. Built with modern web technologies, it simplifies money management, goal tracking, and family coordination into one beautiful, easy-to-use interface.

You can access the app here: [https://juntoo-plum.vercel.app/](https://juntoo-plum.vercel.app/)

## ğŸš€ Getting Started

To run the project locally:

1.  **Prerequisites**: Ensure you have Node.js installed.
2.  **Clone the repository**:
    ```bash
    git clone https://github.com/carlosrincon17/juntoo.git
    ```
3.  **Install dependencies**:
    ```bash
    cd juntoo
    npm install
    ```
4.  **Run the development server**:
    ```bash
    npm run dev
    ```
5.  **Open the app**: Visit [http://localhost:3000](http://localhost:3000) in your browser.
